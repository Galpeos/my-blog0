Modals

Modal auxiliary verbs are used before the base forms of other verbs to add certain meanings to the main verbs of sentences, Modal verbs, or modal verbs, are those verbs in English that help a sentence have a better meaning thanks to their function.

below its are structures:

Affirmative form: subject + modal auxiliary verb + infinitive + complement.

Negative form: subject + modal auxiliay verb + not + infinitive + complement.

Interrogative form: modal auxiliary verb + subject + infinitive + complement?.

List of modals verbs

  • Must: Strong obligation / prohibition / strong necessity / almost absolute certainty
    Example: you must be clean you room

  • Should: suggestions or advice / mild obligation / high level of certainty
    Example: you should be buy a better car

  • May: possibility / permission / polite request
    Example: i had the money, if only may back in time

  • Might: probability
    Example: i think that might be able to lift up that box

  • *Have to: obligation / necessity / lack of necessity (negative form)
    Example: i have to go the train station

  • Could: possibility / past ability / polite request
    Example: you couldn't take that cracker

  • Can: ability / possibility / informal request / impossibility (negative form)
    Example: i can eat one hamburger more

  • Would: invitation / polite request
    Example: if I had studied it probably would have happened
